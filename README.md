# LIME Hub

Este es el Hub de Aplicaciones para LIME (Laboratory Integrated Management Environment).
Es una aplicación estática construida con React, Vite y Tailwind CSS.

## Requisitos

- Node.js (v18 o superior recomendado)

## Instalación

1.  Instala las dependencias:
    ```bash
    npm install
    ```

## Desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

## Construcción

Para generar la versión de producción (estática):

```bash
npm run build
```

Los archivos generados estarán en la carpeta `dist`.

## Estructura

- `src/App.tsx`: Página principal del Hub.
- `src/components/`: Componentes reutilizables.
- `public/Images/`: Imágenes y logos.
